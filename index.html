Инструкция по разархивированию

  1. Скачайте ZIP-файл модификации.
  2. Откройте папку с файлом.
  3. Нажмите правой кнопкой на файл и выберите "Извлечь все...".
  4. Следуйте инструкциям мастера разархивирования.
  5. Откройте папку с распакованными файлами для установки мода или скрипта.
